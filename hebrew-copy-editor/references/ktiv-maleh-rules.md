# Ktiv Maleh (Full Spelling) Rules

## Overview
Ktiv maleh (כתיב מלא) is the standard spelling convention established by the
Academy of the Hebrew Language. It uses matres lectionis (vav and yod) to
indicate vowels, making text readable without nikkud.

## Core Rules

### Vav for /o/ Sound
Add vav (ו) to represent the /o/ vowel:
- תכנה -> תוכנה (software)
- תקשרת -> תקשורת (communication)
- שמרני -> שומרני (conservative)

### Yod for /i/ Sound
Add yod (י) to represent the /i/ vowel:
- שרות -> שירות (service)
- שפור -> שיפור (improvement)
- תעוד -> תיעוד (documentation)

### Double Vav/Yod
When the consonant vav or yod carries a vowel, double it:
- ווילון (curtain) -- consonant vav + vowel vav
- ציון -> ציוון is NOT correct (exception)

## Common Corrections Table
| Ktiv Chaser | Ktiv Maleh | Translation |
|-------------|-----------|-------------|
| תכנה | תוכנה | software |
| שרות | שירות | service |
| תקשרת | תקשורת | communication |
| תכנית | תוכנית | plan/program |
| שפור | שיפור | improvement |
| תעוד | תיעוד | documentation |
| קשור | קישור | link |
| סכום | סיכום | summary |
| חשוב | חישוב | calculation |
| נסוי | ניסוי | experiment |

## Gershayim Rules
- Acronyms: Place gershayim before last letter: צה"ל, מע"מ, בג"ץ
- Abbreviations: Use geresh after the letter: פרופ', ד"ר
- No period after gershayim abbreviations

## Exceptions
Some words are commonly written in ktiv chaser even in formal contexts:
- כל (all) -- not כול
- בן/בת (son/daughter) -- not בין/ביתv
- The definite article ה always attached without vav
